                <li>Communication protocols</li>
                <li>Post-incident review templates</li>
            </ul>
            
            <h3 style="margin-top: 2rem;">Sample Incident Response Flow:</h3>
            
            <div style="background: rgba(0, 0, 0, 0.3); border-radius: 8px; padding: 1.5rem; margin: 1rem 0; border-left: 4px solid #00d4aa;">
                <h4>Phishing Incident Response</h4>
                
                <h5>Detection & Analysis:</h5>
                <ul style="color: #90a4ae;">
                    <li>Monitor email security gateway alerts</li>
                    <li>Analyze email headers and attachments</li>
                    <li>Check URL reputation services</li>
                    <li>Assess potential impact scope</li>
                </ul>
                
                <h5>Containment Procedures:</h5>
                <ul style="color: #90a4ae;">
                    <li>Quarantine malicious emails</li>
                    <li>Block malicious URLs/IP addresses</li>
                    <li>Reset compromised credentials</li>
                    <li>Scan affected systems</li>
                </ul>
                
                <h5>Recovery & Lessons Learned:</h5>
                <ul style="color: #90a4ae;">
                    <li>Conduct user awareness training</li>
                    <li>Update email filtering rules</li>
                    <li>Document incident timeline</li>
                    <li>Implement additional security controls</li>
                </ul>
            </div>
            
            <div style="margin-top: 2rem;">
